# Documentacion Tecnica

Esta carpeta concentra la documentacion vigente del backend y de los flujos compartidos con el frontend.

## Documentacion complementaria del frontend

- `../digital-alert-hub-frontend/README.md`
  Presentacion general del cliente web, setup local y rutas principales.

- `../digital-alert-hub-frontend/docs/README.md`
  Indice tecnico del frontend con seguridad, autenticacion y variables de entorno.

## Documentos disponibles

- `TESTING_GUIDE.md`
  Guia consolidada de pruebas del proyecto, comandos, suites actuales, alcance y brechas pendientes.

- `SECURITY_REPORT.md`
  Informe de seguridad consolidado, estado actual, controles implementados, riesgos residuales y recomendaciones operativas.

- `DEPLOYMENT_DOCUMENTATION.md`
  Guia de despliegue actualizada para backend en Render, frontend en Vercel y base de datos en Neon.

- `REPORTES_IMPLEMENTACION.md`
  Estado funcional y tecnico del modulo de reportes, endpoints, filtros, permisos y componentes principales.

- `ACCOUNT_ACTIVATION_AND_PASSWORD_RECOVERY.md`
  Flujo real de activacion de cuenta, recuperacion de contrasena, OAuth Google y consideraciones de seguridad asociadas.

## Criterio

Esta carpeta es la fuente canonica. Los archivos Markdown ubicados en la raiz del repositorio solo quedan como accesos rapidos hacia estos documentos.
